    Extended description

    I template HTML rendono più semplice l’adozione del modello di sito web e dei servizi digitali per i Comuni Italiani. Il codice HTML/CSS e Javascript dei template è pronto all’uso e già validato in termini di accessibilità e conformità alle Linee guida di design per i servizi web della PA. I template sono costruiti sulle fondamenta dello UI Kit e dalla libreria Bootstrap Italia.

    Sulla pagina del Modello Comuni su Designers Italia è possibile trovare ulteriori risorse a supporto, come la documentazione tecnica e i prototipi a media e alta definizione.
