    Extended description

    La piattaforma OpenDMS è uno strumento per la gestione di un nuovo modello organizzativo e tecnologico, finalizzato alla digitalizzazione di processi propri di una Pubblica Amministrazione e in particolare di un Ateno come l’Università degli Studi di Palermo.

    Il progetto è composto da:

    • un sistema di gestione documentale per una gestione archivistica dei documenti avanzata nel rispetto delle linee guida AgID
    • un Designer Web BPMN 2.0 per la progettazione di processi
    • una integrazione con l’engine BPM Camunda per l’orchestrazione dei processi
    • integrazione con sistemi di protocollo quali ad esempio il protocollo CINECA in uso presso l’Ateneo Palermitano
    • scrivania virtuale fruibile dal web per la gestione delle attività in carico agli utenti e per la consultazione dei documenti condivisi
    • console di posta** per la gestione condivisa e integrata delle caselle PEC o PEO, integrata con il protocollo informatico
