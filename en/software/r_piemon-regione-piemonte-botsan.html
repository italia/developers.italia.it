    Extended description

    BotSan Widget è una webchat automatica utilizzata su alcuni servizi pubblicati sul portale Salute Piemonte capace di interpretare domande poste in linguaggio naturale e di dare una o più riposte pertinenti attraverso l’utilizzo di una piattaforma di Intelligenza Artificiale.

    BotSan Widget è una componente di SOL Assistenza Salute ed è stata realizzata in Node.js usando la React chat [Wolox/react-chat-widget] (https://github.com/Wolox/react-chat-widget)
