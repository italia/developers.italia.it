Ti presentiamo il nuovo Docs Italia!

Un sito unico dove pubblicare i documenti tecnici e amministrativi della Pubblica Amministrazione

Francesco De Augustinis 07 mar 2018
condividi su
  • Twitter

Abbiamo pubblicato una nuova release di Docs Italia, la piattaforma per pubblicare i documenti tecnici e amministrativi della Pubblica Amministrazione in Italia. Un unico sito dove pubblicare documentazione tecnica, linee guida, leggi, circolari e qualsiasi tipo di documento legato ai servizi pubblici o ai progetti digitali della Pubblica Amministrazione.

Scopri il nuovo Docs Italia

  • Come funziona Docs Italia Nella Guida a Docs Italia puoi scoprire come funziona la piattaforma e quali sono i vantaggi e le novità, come il sistema di versionamento o il processo di consultazione pubblica per i documenti amministrativi.

  • Per vedere il nuovo Docs Italia Il modo migliore per scoprire il nuovo Docs Italia è avventurarsi nella homepage ed esplorare la documentazione già pubblicata, come il Codice per l’amministrazione digitale o la documentazione tecnica di ANPR.

  • Partecipa Docs Italia aspetta il tuo contributo! Per iniziare a pubblicare i tuoi documenti su Docs Italia puoi leggere la style guide, che spiega in che formato realizzare i documenti tecnici e amministrativi, e la guida per iniziare a pubblicare.

  • Perché Docs Italia? L’inizio di una rivoluzione Su un post su Medium vi raccontiamo perché è nato il nuovo Docs Italia: un progetto per cambiare il linguaggio dei progetti della Pubblica Amministrazione e renderli più efficaci e trasparenti.

Inizia subito a pubblicare!

Le amministrazioni che vogliono iniziare a pubblicare i propri documenti su Docs Italia possono consultare la guida alla pubblicazione. Scrivici per qualsiasi dubbio o… inizia a pubblicare i tuoi documenti!
