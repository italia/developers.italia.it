    Descrizione estesa

    Abilita un’interfaccia chatbot nel tuo sito Web che comunica con un servizio remoto.

    Dove abilitato l’utente troverà un bottone in basso a destra da cui si aprirà una finestra modale con la chat del servizio.

    C’è un pannello di controllo dove si possono configurare:

    • l’endpoint del servizio remoto
    • le credenziali ed il token del servizio remoto
    • il path dove mostrare il chatbot (altrimenti sarà visibile ovunque)
    • le etichette personalizzabili della chat

    L’interfaccia di chat può essere inoltre facilmente personalizzabile con il CSS nel vostro tema custom.
