    Descrizione estesa

    Plugin per Plone che permette a SOLR di indicizzare un sito Plone per poi poter usare SOLR come motore di ricerca nel sito.

    Comprende i seguenti componenti.

    Control Panel

    Nel pannello di controllo sono disponibili:

    • Flag per abilitare o disabilitare l’integrazione di SOLR
    • Solr URL: URL del core di SOLR
    • i Tipi di contenuto da indicizzare in SOLR
    • URL del frontend pubblico
    • pulsante per forzare l’aggiornamento dell’indice

    Campi aggiuntivi

    Vengono aggiunti a tutti i tipi di contenuto i seguenti campi:

    • Mostra nella ricerca: per poter eventualmente indicare singoli contenuti che non devono essere indicizzati
    • Parole di ricerca: per indicare specifiche parole chiave che fanno sì che quel contenuto abbia un ranking maggiore nei risultati delle ricerche (Elevate).

    Campi del registro nascosti

    Ci sono alcuni campi del registro di servizio che sono nascosti per non permettere agli utenti di modificarli:

    • ready: un flag che specifica se il prodotto è pronto e inizializzato. Indica che lo schema xml è stato caricato.
    • index_fields: è la lista dei campi di SOLR caricati dal file schema.xml

    Caricamento del file schema.xml

    I campi di SOLR sono letti direttamente dal schema.xml esposto da SOLR. Questo schema è salvato nel registry di Plone per questioni di performance ed è sempre sincronizzato quando vengono salvate le configurazioni nel pannello di controllo o si clicca manualmente sul bottone “Ricarica schema.xml”.
